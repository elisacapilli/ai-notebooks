# Verifica automatica e crowdsourced delle informazioni
Repository contiene codice Python sviluppato per tesi magistrale che esplora l'uso di **Large Language Models (LLM)** per supportare i processi di fact-checking.

Il progetto prevede due scenari sulla verifica di 100 affermazioni politiche:
1. **Autonomo**: il modello verifica le informazioni senza input esterni.
2. **Collaborativo**: il modello riceve valutazioni da cinque utenti ottenute tramite crowdsourcing.

L'obiettivo è valutare come il contributo umano possa migliorare le capacità di fact-checking dei LLM open-source.

**Tecnologie utilizzate:** Python, Jupyter Notebook, Pandas, LLMs (Gemma 2B, Mistral B).

Questo progetto dimostra competenze in programmazione Python, gestione dati, applicazioni di AI e analisi collaborativa.
